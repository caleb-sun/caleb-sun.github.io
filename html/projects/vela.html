    Tools

    Vela Projects is a construction consulting firm offering Project Management and Owner's Representative Services. Their mission is to help clients successfully navigate the complexities of building projects.

    Branding

    I was initially brought on to refine and refresh the existing branding. Vela already had a logo with a personal story to the owner, so I took a conservative approach to refine the logo geometrically, as well as create a comprehensive branding package compatible to the client's Google-based platform.

    Materials

    The next phase of the project was the create branded materials for the client's needs, recreating existing documents with the new branding.

    Website

    The final phase of this project is to design a website based on the client's needs, including setting up the site for further managable maintainance. For this reason, we used Wix. The website was first mocked up on Figma, and then transfered onto Wix with dynamic content.
